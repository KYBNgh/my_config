## My Config
This is my config. So far, they are just about bash and readline(.inputrc).

## Usage
```bash
1. Clone this repo to a directory:
git clone https://github.com/gnuunixchad/dotfiles

2. Move files:
mv .config/bash ~/.config/bash
mv .inputrc ~/.inputrc

3. Create symlinks:
ln -s ~/.config/bash/bash_profile.sh ~/.bash_profile
ln -s ~/.config/bash/bashrc.sh ~/.bashrc

4. [Optional] Remove this clone:
cd ..
rm -rf my_config/

```
